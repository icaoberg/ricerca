Ricerca for Python 1.0
======================
Ricerca is a python implementation of FALCON as described in 
Leejay Wu, Christos Faloutsos, Katia P. Sycara, and Terry R. Payne. 2000. FALCON: Feedback Adaptive Loop 
for Content-Based Retrieval. In Proceedings of the 26th International Conference on Very Large Data Bases 
(VLDB '00), Amr El Abbadi, Michael L. Brodie, Sharma Chakravarthy, Umeshwar Dayal, Nabil Kamel, Gunter 
Schlageter, and Kyu-Young Whang (Eds.). Morgan Kaufmann Publishers Inc., San Francisco, CA, USA, 297-306.


Pre-Requisites
==============
-numpy, scipy

Installation
============
Run
> sudo python setup.py install
